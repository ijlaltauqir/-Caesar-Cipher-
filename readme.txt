This repository contains a Python-based command-line tool
 that implements both Caesar and Vigenère ciphers as part of a beginner-friendly cybersecurity project.
  It allows users to easily encrypt and decrypt text using a shift key (Caesar) or a keyword (Vigenère),
   while also demonstrating how weak encryption can be broken. 
   The tool includes features like brute-forcing all possible Caesar shifts and automatically 
   cracking Caesar-encrypted text using English letter frequency analysis.
    It also supports processing entire text files, making it more practical to use.
     Built using only Python’s standard library, this project helped me understand important concepts like modular arithmetic, character encoding with ord() and chr(), file handling, and basic cryptanalysis.
      Overall, it serves as a simple but effective learning project to explore the foundations of cryptography and cybersecurity.
